Conectar planilha

Publique a planilha no Google Sheets (não é a mesma coisa que compartilhar) e cole aqui o link CSV gerado. A partir disso o painel atualiza sozinho, sem precisar editar este arquivo de novo.

  1. Abra a planilha no Google Sheets → Arquivo → Compartilhar → Publicar na web
  2. Em "link", selecione a aba correta e escolha o formato CSV
  3. Clique em Publicar e copie o link gerado
  4. Cole o link no campo abaixo
